# [Статистика](statistics)

Для быстрого доступа к странице «Статистика» из любого раздела нажмите на иконку в боковом меню слева.

![Sidebar for Statistics(95%)](https://img.solarspace.pro/docs/saas/statistics/statistics-full.png "Боковое меню раздела «Статистика»")

На странице отображаются отчёты по всем вашим ресурсам. Доступны 4 вкладки:

![All tabs for Statistics(95%)](https://img.solarspace.pro/docs/saas/statistics/pages-for-statistics.png "Все вкладки раздела «Статистика»")

- [**Ширина канала**]([236])
- [**Запросы**]([237])
- [**Коды ответов**]([238])
- [**География запросов**]([239])

Система предоставляет детальную аналитику по защитным функциям.

В модулях **WAF Lite** и **Antibot** доступна статистика:

- **Проанализированные запросы** — общее количество запросов, прошедших проверку.
- **Валидные запросы** — запросы, успешно допущенные к обработке.

![Statistics services tabs(95%)](https://img.solarspace.pro/docs/saas/statistics/pages-for-statistics-services.png "Вкладки статистики по защитным функциям")

Все виды статистики доступны в едином интерфейсе для удобного мониторинга работы сервиса.
