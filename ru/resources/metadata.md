# [Мои сайты](my-resources-page)

На странице «Мои сайты» отображается список всех созданных вами сайтов.

Для быстрого доступа к странице «Мои сайты» из любого раздела кликните на иконку в виде папки в боковом меню.

![My sites page on sidebar (95%)](https://img.solarspace.pro/docs/saas/resources/resources-on-sidebar.png "Переход на страницу «Мои сайты»")

Если у вас ещё нет сайта под защитой Solar Space, выполните шаги из [Быстрого старта]([300]) — в инструкции подробно описано, как добавить сайт и подключить домен к защите. 

В Solar Space один сайт соответствует одному домену. Если нужно подключить к защите ещё один домен, создайте для него отдельный сайт на странице [«Мои сайты»](https://my.rt-solar.ru/resources).

![Add site on My sites page (95%)](https://img.solarspace.pro/docs/saas/resources/my-sites-add.png "Кнопка «Добавить сайт» на странице «Мои сайты»")


На странице "Мои сайты" можно:
- Отфильтровать список по статусу через выпадающий список "Статус".
- Найти нужный сайт через строку поиска.
- Посмотреть домен и статусы подключенных сервисов в строке сайта.
- Открыть меню действий по сайту через кнопку с тремя точками.

![Menu on resources page(95%)](https://img.solarspace.pro/docs/saas/resources/menu-on-resources-page.png "Меню на странице ресурсов")

Из меню действий доступны разделы сайта:
- [Настройки]([249])
- [Статистика]([247])
- [Сервисы]([248])

Чтобы перейти к настройкам, статистике или управлению сервисами конкретного сайта, откройте меню действий `«⋯»` в строке сайта и выберите нужный раздел.
