## [Тақырыптар оқиғасында](heading-in-table-of-contents)
Тақырыпты оқиғаға қосу үшін тақырыпқа сілтеме қосу керек. Сілтеменің орнында басқа белгілер болса, олар есептелмейді.

Оқиғаға қосылған тақырыптың құрылым схемасы:
```md
# [Тақырыпта көрсетілетін мазмұн](тақырыпқа-сілтеме "Оқиғадағы атауы")
```

- `Тақырыпта көрсетілетін мазмұн` мазмұндың көрсетілетін тақырып ретінде көрсетіледі;
- `тақырыпқа-сілтеме` оның навигациясы үшін пайдаланылады және тақырыпқа скроллдану кезінде URL-дағы hash-ты өзгертеді;
- `Оқиғадағы атауы` - міндетті емес бөлім. Оқиғада көрсетілетін мазмұннан айырылған атау пайдаланылады.
Оны қойсаңыз, оқиғада `Тақырыпта көрсетілетін мазмұн` деп көрсетіледі.

<br/>

## [Пайдалану](using)
Бетте тек 1-денгей тақырып болуы керек, ал алдағы беттің оқиғасы дұрыс жасалмайды.
Егер кейбір тақырыптарды оқиғада көрсету қажет емес болса, онда:
```
# [Тақырыпта көрсетілетін мазмұн](тақырыпқа-сілтеме "Оқиғадағы атауы")
```
келесін пайдалануға болады:
```
# Тақырыпта көрсетілетін мазмұн
```
Осы арқылы тақырыптың деңгейін таңдауға болады.
