# Суреттер
Осы бөлімде суреттер мен оларды қалай қолдану туралы талқыланады.

<br/>

## Сипаттама
Суретті жасау үшін синтаксис пайдаланылады:
```md
![Суреттің атауы](суреттің мекенжайы "Суреттің жасырын мәтіні")
```
- `Суреттің атауы` бұл көрсетілетін мәтін, суретті жүктеу кезінде қате пайда болғанда пайда болады;
- `суреттің мекенжайы` бұл кескіннің url мекенжайы;
- `Суреттің жасырын мәтіні` бұл суретке курсорды қойғанда көрсетілетін мәтін.

<br/>

## Пайдалану мисалы
```md
![сурет](https://placehold.co/1600x2500 "Кейбір мәтін")
```
