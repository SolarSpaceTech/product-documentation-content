# Комбинированные тізімдер
Бұл мақалада орналасқан болмасақ және тізімдерді қолдануға үлесі берілген мисал көрсетілген.

<br/>

## Пайдалану мисалы

```md
Элементтер түрлері:
- Блоктық:
  - Тақырып
    1. 1-дәрежелі
      2. 2-дәрежелі
      3. 3-дәрежелі
      4. 4-дәрежелі
      5. 5-дәрежелі
      6. 6-дәрежелі
  - Параграф
  - Тәртіпті және тәртіпсіз тізімдер
  - Бөліктірік
  - Отступ
  - Блок код
  - Сурет
  - Ақпараттық блок
  - Кесте

- Сызықты
  - Сілтеме
  - Көлбеу мәтін
  - Үлкен мәтін
  - Сызық код

- Басқалар:
  - HTML пішімі
```
