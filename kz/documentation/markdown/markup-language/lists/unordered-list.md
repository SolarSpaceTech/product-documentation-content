# Таңдалмаған тізімдер
Осы мақалада таңдалмаған тізімдер мен оларды қолдану талқыланады.

<br/>

## Сипаттама
Таңдалмаған тізімді жасау үшін жолдың басында `-` пайдалану керек. Тізімнің көшірмесі кездейсоқ `-` басылуы керек. Көптілді таңдалмаған тізім жасау үшін `-` белгісінен алдында 3 бос орын қолдану керек, келесі деңгей 3-ге бөлінген бос орын санымен жылжыту керек.

<br/>

```md
Элементтер түрлері:
- Блоктық:
  - Тақырып
  - Параграф
  - Тізімдер
  - Бөлікшелдіріш
  - Отступ
  - Блок код
  - Сурет
  - Ақпараттық блок
  - Кесте

- Сызықты
  - Сілтеме
  - Көлбеу мәтін
  - Қалыпты мәтін
  - Сызық код

  - HTML пішімі
```
