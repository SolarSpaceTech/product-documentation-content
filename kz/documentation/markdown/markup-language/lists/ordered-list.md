# Орналасқан тізімдер
Осы мақалада орналасқан тізімдер мен оларды қолдану талқыланады.

<br/>

## Сипаттама
Орналасқан тізімді жасау үшін өзіңізге керек болатын санды бастап алу керек `(сан).` жолдарында. Әр бір жаңа тізім элементі
`(сан).` басталуы керек. Мысалы, көп қатарлы орналасқан тізім үшін `(сан).` алдында 3 бос орын қолдану керек,
кейінгі деңгейдегі өзгеру 3-ге бөлінген бос орындармен айналуы керек.

<br/>

## Қолдану мисалы
```md
Элементтер түрлері:
1. Блоктық:
  1. Тақырып
    1. 1-денгей
    2. 2-денгей
    3. 3-денгей
    4. 4-денгей
    5. 5-денгей
    6. 6-денгей
  2. Параграф
  3. Орналасқан және орналаспаған тізімдер
  4. Бөлікшелдіретін
  5. Отступ
  6. Блок код
  7. Сурет
  8. Ақпараттық блок
  9. Кесте

2. Сызықты
  1. Сілтеме
  2. Көлбеу мәтін
  3. Қалың мәтін
  4. Сызық код

3. Басқалар:
  1. HTML пішімі
```
