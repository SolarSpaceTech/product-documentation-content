# Сілтемелер
Осы бөлімде сілтемелер мен оларды қалай қолдану туралы талқыланады.

<br/>

## Сипаттама
Сілтеме құру үшін синтаксис пайдаланылады:
```md
[Сілтеме мәтіні](сілтеменің/барады/жол "Сырынды мәтін")
```
- `Сілтеме мәтіні` көрсетілетін мәтін;
- `сілтеменің/барады/жол` url немесе байланысты жол;
- `Сырынды мәтін` сілтеме бойынша сол көрсетілетін мәтін.

Егер url толық мекенжайымен `http` немесе `https` протоколдерін қосау керек болса, оған жаңа браузер терезесі ашылады,
басқа жағдайда қазіргі терезеде өтіледі.

<br/>
