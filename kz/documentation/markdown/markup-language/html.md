# HTML
Осы бөлімде Markdown-жасауда HTML пайдалану туралы сипаттама берілген.

<br/>


## Сипаттама
HTML ретінде қолдануға болатын барлық жасақтама қолдануға болады, бірақ, егер Markdown-ді HTML-ге ауыстыру үшін ағымдағы движокты қолдану мүмкін болса, онда оны қолдану керек. Мысалы, стандартты HTML кескін тегін пайдалансаңыз, ол дұрыс түрде құрылмайды және кескінді көрсетушін іске қосу мүмкін емес.

<br/>


## Пайдалану мисалы

```md
<h4></h4>
<p></p>
<img src="https://placehold.co/400x600" alt="Заглушка"/>
<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Row 1 - Column 1</td>
    <td>Row 1 - Column 2</td>
  </tr>
  <tr>
    <td>Row 2 - Column 1</td>
    <td>Row 2 - Column 2</td>
  </tr>
</table>
<br>
```
