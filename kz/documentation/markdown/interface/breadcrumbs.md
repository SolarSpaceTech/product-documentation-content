# Тығыз екмек документациясы

Тығыз екмек — басты беттен ағымдағы мақалаге жолдама беретін интерфейс элементі. Бұл интерфейс элементі пайдаланушыларға документацияның құрылымын түсіну
көмектеседі. Ол беттің жоғарғы бөлімінде, шапка және мақала арасында көрсетіледі.

![Тығыз екмек](https://raw.githubusercontent.com/SolarSpaceTech/product-documentation-content/refs/heads/main/ru/documentation/markdown/images/breadcrumbs.png)

<br/>

## [Негізгі функциялар](basic-functions)

- **Иерархия бойынша навигация:** Басты беттен ағымдағы мақалаге жолдама беретін жолдарды көрсетеді, бөлімдердің және астардың құрылымын көрсетеді.
- **Орын анықтамасының жақсартуы:** Пайдаланушыларға документацияның бүтін құрылымынан қайда орналасқанды түсіндіреді.
- **Пайдаланушы тәжірибесін жақсарту:** Ұзын беттерді қарау кезінде пайдаланушының "ҹабиттену" деңгейін азайтады.

<br/>

## [Мета-сипаттарды пайдалану арқылы теңшеу](customization-using-meta-properties)

Документацияда тығыз екмектерді теңшеу үшін келесі мета-сипаттарды пайдалану:

- **`displayName`:** Бөлім немесе мақала атауын тығыз екмектерде көрсетеді.
  ![displayName сипатының әсері](https://raw.githubusercontent.com/SolarSpaceTech/product-documentation-content/refs/heads/main/ru/documentation/markdown/images/display-name.png)

<br/>

## [Мета-сипаттардың әсері](impact-of-meta-properties)

- `displayName` тығыз екмектерде бөлімдер мен мақалалардың атауларын көрсету үшін пайдаланылады


<br/>

## [Орнату процессі](step-by-step-setup-guide)

### Кадам 1: Документациялық құрылымды белгілеу

- Бөлімдер мен мақалалардың үлгілері мен файлдар арқылы документацияны файл жүйесінде түрлендіріңіз.

### Кадам 2: Файлдарда мета мүшелерді орнату

**Бөлім мета деректерінің мисалы:**

```md
---
displayName: Нұсқаулар
order: 1
published: true
---
```

**Мақала мета деректерінің мисалы:**

```md
---
title: Продуктты орнату
displayName: Орнату
order: 1
published: true
---
```

### Кадам 3: `displayName` орнату үшін

- `displayName`-де түсіндірмелі және қысқа атауларды белгілеу арқылы аспаптың ақпараттық және ауыр мекенжайы болуын көрсетіңіз.

### Кадам 4: Нұсқауларды тексеру

- Документацияны жасау және көрсетілген мақалаларға өту арқылы нұсқауларды дұрыс көрсетілгенін тексеріңіз.
- Жолды дұрыс көрсетілгенін тексеріңіз.

<br/>

## [Ұсыныстар](recommendations)

- **Атаулардың қысқа болуы:** `displayName`-де қысқа және түсіндірмелі атауларды пайдаланыңыз, аспапта дұрыс көрсетілуі үшін.
- **Стильдік біріктілігі:** Барлық документациядағы атаулардың жазбалы стилін ұстаңыз.
- **Ішкі ұрпақтардан сақталу:** Жолды өте дерін жасауға тырысу, аспаптардың ұзындығын арттырудан бас тартыңыз.
- **Көрініс тексеру:** Жолдың қайсы бөлімдері мен мақалаларын көрсетілгенін регулярлық түрде тексеріңіз, кейбір жасырын немесе қажетсіз элементтерді алып тастау үшін.

<br/>

## [Пайдалану мисалы](examples)

**Файл жүйесі:**

- `/kk`
  - `metadata.md` (Қазақша)
  - `/documentation`
    - `metadata.md` (Документация)
    - `/getting-started/`
      - `metadata.md` ("Бастау")
      - `installation.md` ("Орнату")
      - `configuration.md` ("Орнату") 

**"Бастау" бөлімі үшін мета деректері:**

```md
---
displayName: Бастау
order: 1
published: true
---
```

**"Орнату" мақаласы үшін мета деректері:**

```md
---
title: Продуктты орнату
displayName: Орнату
order: 1
published: true
---
```


**"Орнату" бетіндегі "Орнату" бетіндегі ақпаратты көрсету:**

```
Документация / Бастау / Орнату
```

## [Жалғастыру](conclusion)

Хлебные крошки документацияда навигациялық элемент болып саналады, пайдаланушылардың ориентациясын жақсартуға көмектеседі және сайтта навигацияны оңайлауға көмектеседі. `displayName` мета қасиетін дұрыс орнату арқылы хлебнің крошкаларын дұрыс орнату, көрсетілетін және әрекетті қолайландыруға мүмкіндік береді.
