# Документация элементтерінің сипаттамасы

Мета-сипаттамалар арқылы документация элементтерінің көрсетуін жиі әдістемелі және жиі көшірмелерді басқаруға мүмкіндік береді. Олар арқылы сіз ашықтарды күрделеу мүмкіндіктерін күрделеу мүмкін:

- **[Басты бет]([9])**

  - **`category`**: Мақаланы көрсету үшін басты беттің бөлігін белгілейді (`start`, `services`, `useful`, `faq`).
  - **`categoryName`**: Басты беттің бөлігінде мақаланың атауын көрсетеді.
  - **`categoryDescription`**: Бөліктің мақаласына түсіндірме береді.
  - **`categoryOrder`**: Бөлік ішіндегі мақалаларды көрсету ретін белгілейді.
  - **`categoryIcon`**: Бөліктең мақаласы үшін белгілікті көрсетеді.

- **[Жоғары панель]([7]) (Header)**

  - **`headerName`**: Панельдегі мақаланың атауын белгілейді.
  - **`headerOrder`**: Панельдегі сілтемелерді көрсету ретін белгілейді.

- **[Төменгі панель]([6])**

  - **`footerName`**: Панельдегі мақаланың атауын белгілейді.
  - **`footerOrder`**: Панельдегі сілтемелерді көрсету ретін белгілейді.

- **[Мәзір]([10])**

  - **`displayName`**: Мәзірде бөліктер мен мақалалардың атауын белгілейді.
  - **`order`**: Мәзірде бөліктер мен мақалалардың көрсету ретін белгілейді.

- **[Шығу қуылымдары]([4])**

  - **`displayName`**: Шығу қуылымдарында бөліктер мен мақалалардың атауын белгілейді.

- **[Мақала мазмұны]([11])**

- **[Кіру тарихы]([8])**

  - **`historyName`**: Кіру тарихында мақаланың атауын белгілейді.
  - **`historyDescription`**: Кіру тарихында мақаланың түсіндірмесін береді.

- **Мазмұнның көрінісі**

  - **`published`**: Бөліктер мен мақалаларды сайтта көрсетуін басқарады. `false` ретінде орнатылса, элементтер мәзірде, жоғары панельде, төменгі панельде,
    басты бетте және кіру тарихында көрсетілмейді.

- **Бетбақтың тақырыбы**
  - **`title`**: Браузердің тақырыптық тақтасында көрсетілетін мәтінді анықтайды.

Бұл мета қасиеттерді пайдалана отырып, пайдаланушыларға қолайлы навигация және мақсатты бөлімдерге қатынасу арқылы ұсынымды документация интерфейсін теңшеуге болады.
