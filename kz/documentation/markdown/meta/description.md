# Сипаттама

Мета тақырыптар HTML-беттерді Markdown файлдарынан құрылған файл жинасына негізделген HTML-беттерді жасау кезінде кіру-шығуларды басқару үшін кілті құралдарды талдайды.
Олар документацияның мазмұнын көрсету және ұсыныстарды басқару аспекттерін басқару үшін қолданылады.

<br/>

| Тақырып               | Сипаттама                                                                                                                                  | Деректер түрі және мүмкін болатын мәндер                                         | Міндеттілік | Орындалған |
|:----------------------|:------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------|:-----------:|:-----------:|
| `id`                  | Тіл ішіндегі мақаланың бірден-біреуі; қысқа сілтемелер мен кіру-шығу тарихы үшін пайдаланылады                                            | Сан                                                                              |       +     |      -      |
| `title`               | Мақала атауы, браузердің тақырыптық тақтасында көрсетілетін атау                                                                       | Строка                                                                           |       +     |      +      |

| `order`               | Ағымдағы бөлім ішіндегі мақаланың тәртібін белгілейді                                                                                 | Сан                                                                              |       +     |      +      |

**Бағандар түсіндірмесі:**

- **Мүлкіет**: Мақала мета-мүлкіетінің атауы.
- **Сипаттама**: Мүлкіеттің мақсаты туралы қысқаша сипаттама.
- **Деректілер түрі және мүмкін болатын мәндер**: Күтілетін деректілер түрі және қабылдайтын мәндер.
- **Міндеттілік**: Мета-мүлкіетті пайдалану қажеттілігі.
- **Орындалды**: Мүлкіет орындалу мүмкіндігінің ағымдағы мәртебесі.
