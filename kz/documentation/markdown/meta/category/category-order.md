# categoryOrder Сипаттамасы

**Мета қасиеті:** `categoryOrder`

**Деректер түрі:** Сан

**Есікті өзгерту аумағы:**
![Қасиеттің өзгеруі](https://raw.githubusercontent.com/SolarSpaceTech/product-documentation-content/refs/heads/main/ru/documentation/markdown/images/category-order.png)

<br/>

## [Сипаттама](description)

Мета қасиеті `categoryOrder` сайттың басты бетіндегі белгілі бөлім ішіндегі мақаланы көрсету ретінде көрсету порядокты белгілеу үшін пайдаланылады.
Ол мақалаларды бөлім ішіндегі дәлелдерді дұрыс теңестіру арқылы мақалалардың тізімін баптауға мүмкіндік береді, көрсету порядоктың келесі мақаланы көрсетуін бақылау.

<br/>

### [Негізгі функциялар](basic-functions)

- **Мақалаларды бөлім ішіндегі порядокпен басқару:** Басты беттегі бір бөлім ішіндегі мақалаларды қажетті тізімдеу мүмкіндігін береді.
- **Көрсету үстелігі:** `categoryOrder` мәні кем болатын мақалалар бөлім тізімінде жоғарыда көрсетіледі, оларға жоғары үстелік береді.

<br/>

### [Басқа мета қасиеттермен қалыпты іске асыру](with-other-properties)

- **`category`:** Мақаланы көрсету керек басты беттең бөлімін анықтайды. `categoryOrder` тек бір және оған тең мақалаларға қолданылады.
- **`categoryName`:** Мақаланың бөлім бетінде көрсетілетін атауын белгілейді.
- **`categoryDescription`:** Мақаланың бөлім бетінде көрсетілетін түсіндіруін береді.
- **`categoryIcon`:** Мақаланың атауының жанында көрсетілетін суреттің жолын көрсетеді.

<br/>

### [Мағынаны ақпараттар](notes)

- **Міндеттілік:** `categoryOrder` мақаланы бірін бір бөлімдегі көрсету үшін анықталған болса, міндеттіді.
- **Мәндердің уникалдығы:** `categoryOrder` мәндерін бір бөлім ішінде уникалды мәндерді пайдалану ұсынылады, көрсету порядокты баптау үшін.
- **Мәндер ауқымы:** Кез келген бүтін сан мәндерін қабылдауға болады. Қарапайымдылығын басқару үшін қолдану үшін қарапайым сандарды пайдалану мақұлдық.


<br/>

### [Қолдану кезінде](when-to-use)

- **Последовательдік бақылау:** Мақсатты статейлерді белгілі бағытта көрсету маңызды болған кезде, мисалы, кіріспе материалдардан көп қиындықтарға дейін.
- **Маңыздылықты белгілеу:** Негізгі немесе жаңа статейлерді басқа секцияда жою арқылы ерекше немесе жаңа статейлерді белгілеу үшін.
- **Мазмұнды ұйымдау:** Пайдаланушыларға ақпаратты логикалық топтау және көрсету ретінде белгілеу үшін.

<br/>

### [Пайдалану сұраулары](advice)

- **Жоспарлау:** Секцияның ұқсасын және статейлерді бөліп тарату үшін `categoryOrder` мәндерін анықтау үшін алдымен жоспарлау.
- **Топтау:** Секцияда қатарлар бар болса, көрсетілген қатарға көрсетілген мәндер аралығын пайдалануға болады (мысалы, 10-19 бір топқа, 20-29 басқа топқа).
- **Жаңа статейлерді қосу:** Жаңа статейлерді қосу кезінде `categoryOrder` мәндерінің логикалық бақылауын сақтауға тырысу.
- **Инкрементті қадамдар:** Мысалы, 10, 20, 30 дейінгі мәндер арасында қадамдарды пайдаланыңыз, жаңа статейлерді барлық болмаса да барлық мәндерді қайта белгілеу қажеті жоқ.
- **Бірдейліктілік тексеру:** Көрсетілген секция ішінде `categoryOrder` мәндерінің бірдейлігін тексеріңіз, көрсету ретінде белгілеудің белгісіздігін алу үшін.
- **Ойланған бағыт:** Статейлердің бағытын олардың маңыздылығы және оқу ретімен сәйкес келетін бағыттарға сәйкес пландау.

<br/>

## [Жалғастыру](conclusion)

`categoryOrder` мета қасиеті басқару панелінде секциялардағы статейлерді көрсету ретін белгілеу үшін кінәлі құралдың бірі болып табылады.
Ол пайдаланушылардың сайттан навигациясын жақсарту, барлық пайдаланушылардың сайтты пайдалану қиындығын жақсартуға мүмкіндік береді. 
Осы қасиетті дұрыс пайдалану сайттың мазмұнды ұйымдауға көмектеседі және ресурсты пайдалану ыңғайлылығын арттырады.

<br/>
