# categoryName Қасиеті

**Мета қасиет:** `categoryName`

**Деректілік түрі:** Строка

**Есіктік аударым:**
![Қасиеттің әсері](https://raw.githubusercontent.com/SolarSpaceTech/product-documentation-content/refs/heads/main/ru/documentation/markdown/images/category-name.png)


<br/>

## [Сипаттама](description)

Мета қасиеті `categoryName` сайттың басты бетінде көрсетілетін мақаланың атауын анықтау үшін пайдаланылады. 
Бұл атау көрсетілетін секцияда пайдаланушыларға көрсетіледі (`start`, `services`, `useful` немесе `faq` қасиет арқылы анықталады),
басты бетті көретін уақытта.

<br/>

### [Негізгі функциялар](basic-functions)

- **Басты бет атауын орнату:** Басты бетте тек атауын қайтарады, мақаланың басты беттегі атауынан (displayName) айырмашылықты анықтау мүмкіндігі бар атауны белгілейді.

<br/>

### [Басқа мета қасиеттермен қалай жұмыс істейді](with-other-properties)

- **`category`:** Мақаланың қай секцияда көрсетілетінін анықтайды.
`start`, `services`, `useful`, `faq` мүмкіндіктерінен бірін қабылдайды.
- **`categoryDescription`:** Басты бетте көрсетілетін мақаланың қысқаша сипаттамасын береді.
- **`categoryOrder`:** Мақаланың басты беттегі секцияда көрсетілу ретін белгілейді.
- **`categoryIcon`:** Мақаланың атауының жанында көрсетілетін суреттің жолын көрсетеді.
- **`displayName`:** Мәзірде және нанымдагыда пайдаланылатын мақаланың барлық атауы. `categoryName` және `displayName` айырмашылығы болуы мүмкін.

<br/>

### [Маңызды ескертулер](notes)

- **Мақаланы басты беттегі секцияда көрсету үшін `category` және `categoryName` көрсету міндетті:**.
- **`categoryName` жоқ болуы:** Егер `category` көрсетілген болса, бірақ `categoryName` жоқ болса, көрсетілетін мән `displayName` мәнін пайдаланады.
- **Атауның уникальдігі:** `categoryName` ақпараттың анық және уникальді болуы үшін, секция контекстінде айырмашылығын алу үшін ұсынылады.


<br/>

### [Не қолдану кезінде](when-to-use)

- **Басты бет үшін атауын адаптациялау:** Статияны басты бет үшін толықтыру қажет болған кезде.
- **Атаулардың айырмашылығы:** Статияның атауы менюде немесе тақырыпта тым ұзын немесе техникалық болған кезде.

<br/>

### [Пайдалану сұрақтары](advice)

- **Қолайлық:** `categoryName` статияның мазмұнымен сәйкес келетінін тексеріңіз және пайдаланушыларды алдап алмасын.
- **Тексеру:** `categoryName`-ді `categoryDescription` және белгішесімен бірге басты бетте қалай көрінедігін тексеріңіз.

<br/>

## [Жалғастыру](conclusion)

`categoryName` мета түйіні басты бетте статияны көрсету үшін маңызды құралдық. Бұл пайдаланушыларға документацияға кіруге қызығушы және әдемі түрде мазмұнды көрсету мүмкіндігін береді.

<br/>

## [Пайдалану мисалы](examples)

```md
---
title: Толық орнату нұсқаулығы
displayName: Орнату нұсқаулығы
order: 5
published: true
category: services
categoryName: Жылдам орнату
categoryDescription: Біздің сервисті жылдам орнату туралы мәлімет алыңыз
categoryOrder: 1
categoryIcon: /icons/setup.svg
---
# Толық орнату нұсқаулығы

Біздің сервісті жылдам орнату туралы толық нұсқаулыққа қош келдіңіз...
```

Мұнда:
- **`category`:** Статия `services` бөлімінде көрсетіледі.
- **`categoryName`:** `services` бөлімінде статия "Жылдам орнату" атауымен көрсетіледі.
- **`categoryDescription`:** Атау астында "Біздің сервисті жылдам орнату туралы мәлімет алыңыз" түсіріледі.
- **`categoryOrder`:** Статия `services` бөлімінің тізімінде бірінші болады.
- **`categoryIcon`:** Атау астында `/icons/setup.svg` белгісі көрсетіледі.
```
